## Getting Started
> To get started:
> clone the repo...

  `git clone https://github.com/hamid-murambiwa/Leaderboard.git`

> navigate to repo dir...

  ```cd toDoList```

> install project dependencies

  ```npm install```

> build project

  ```npm run build```

> start project

  ```npm start```
